# =============================================================================
# PARAMETER GUIDELINES
# =============================================================================

# nWifis (Number of nodes):
#   - Small: 5-10 nodes
#   - Medium: 10-20 nodes
#   - Large: 20-40 nodes
#   - Constraint: nWifis > nSinks

# nSinks (Number of receivers):
#   - Typically 20-40% of total nodes
#   - More sinks = more simultaneous flows

# totalTime (Simulation duration in seconds):
#   - Short tests: 30-60 seconds
#   - Standard: 100-150 seconds
#   - Long tests: 200-300 seconds
#   - Longer times give more stable statistics

# rate (Traffic generation rate):
#   - Low: 256-512 bps
#   - Medium: 1024-2048 bps
#   - High: 4096-8192 bps

# nodeSpeed (Maximum speed in m/s):
#   - Pedestrian: 1-5 m/s
#   - Urban vehicle: 10-20 m/s
#   - Highway vehicle: 25-40 m/s
#   - Note: 1 m/s â‰ˆ 3.6 km/h

# nodePause (Pause time in seconds):
#   - No pause: 0 seconds (continuous movement)
#   - Short pause: 2-5 seconds
#   - Medium pause: 5-10 seconds
#   - Long pause: 10-20 seconds
#   - Higher pause = more stable topology
